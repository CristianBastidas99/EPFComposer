Task: Identificar los requisitos de negocio para el proceso de negocio
Identificar y listar los requisitos funcionales y no funcionales que el proceso de negocio debe cumplir.
Purpose
Identificar y documentar los requisitos específicos del negocio que deben ser satisfechos por el proceso de negocio.
Relationships
RolesPrimary Performer:
  • Analista de Negocio

    		Optional:

    • Definición de proceso de negocio
      Main Description
      En esta tarea, se analizarán las necesidades y expectativas del negocio y se identificarán los requisitos específicos que el proceso de negocio debe satisfacer. Estos requisitos pueden estar relacionados con funcionalidades, rendimiento, seguridad, escalabilidad, entre otros aspectos relevantes. Al identificar los requisitos de negocio para el proceso de negocio, se establecen las metas y criterios que deben ser alcanzados para que el proceso sea exitoso.
      Steps
      Revisar la definición de proceso de negocio para comprender su propósito y alcance.
      Identificar los objetivos del negocio que se espera que se logren con el proceso de negocio.
      Identificar los requisitos funcionales y no funcionales para el proceso de negocio, como los flujos de trabajo, los roles de usuario, los informes y las regulaciones aplicables.
      Documentar los requisitos del proceso de negocio y validarlos con el equipo de proyecto y los interesados.
