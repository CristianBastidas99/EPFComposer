Task: Verificar que los requisitos estén correctamente relacionados en la matriz de trazabilidad
Revisar y confirmar que cada requisito funcional esté adecuadamente vinculado en la matriz de trazabilidad con los casos de uso, pruebas y otros elementos relevantes.
Purpose
Comprobar que los requisitos funcionales estén correctamente trazados y relacionados en la matriz de trazabilidad.
Relationships
RolesPrimary Performer:
  • Analista de requerimientos
Additional Performers:
InputsMandatory:
  • Análisis de requisitos existentes y relaciones entre artefactos
    		Optional:
    • None
    Outputs
    • Matriz de trazabilidad de requisitos
      Main Description
      En esta tarea, se realizará una verificación exhaustiva de la matriz de trazabilidad para asegurarse de que todos los requisitos funcionales estén correctamente relacionados con los demás elementos del proyecto, como los casos de uso, pruebas, componentes y otros requisitos. Esto garantiza la coherencia y la trazabilidad de los requisitos en todo el proceso de desarrollo y facilita la gestión y el seguimiento de su implementación.
      Steps
      Revisar la lista de requisitos funcionales del sistema y la matriz de trazabilidad.
      Verificar que cada requisito esté relacionado con los componentes del sistema y las pruebas correspondientes en la matriz de trazabilidad.
      Identificar cualquier requisito que no esté correctamente relacionado y actualizar la matriz de trazabilidad.
      Asegurarse de que la matriz de trazabilidad esté completa y precisa para garantizar que todos los requisitos estén cubiertos por las pruebas y los componentes del sistema.
