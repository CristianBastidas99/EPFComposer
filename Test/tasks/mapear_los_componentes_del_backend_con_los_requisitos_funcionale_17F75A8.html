Task: Mapear los componentes del BackEnd con los requisitos funcionales
Identificar y relacionar los componentes del backend con los requisitos funcionales que son responsables de satisfacer.
Purpose
Establecer la relación entre los componentes del backend y los requisitos funcionales del sistema.
Relationships
RolesPrimary Performer:
  • Desarrollador BackEnd
Additional Performers:
InputsMandatory:
  • Matriz de Rastreabilidad
    		Optional:
    • None
    Outputs
    • Matriz de Rastreabilidad
      Main Description
      En esta tarea, se realizará un mapeo entre los componentes del backend del sistema y los requisitos funcionales que están asociados a ellos. Cada componente del backend debe estar vinculado a los requisitos que debe cumplir para el correcto funcionamiento del sistema. Esto permite tener una visión clara de qué componentes son responsables de satisfacer cada requisito funcional y garantiza que todos los requisitos estén cubiertos por los componentes correspondientes. El mapeo de los componentes del backend con los requisitos funcionales es esencial para comprender cómo se implementa la funcionalidad del sistema.
      Steps
      Analizar los requisitos funcionales del sistema proporcionados por el Analista de Requerimientos.
      Identificar los componentes del BackEnd que están relacionados con cada requisito funcional.
      Mapear cada requisito funcional con los componentes correspondientes en la Matriz de Rastreabilidad.
      Verificar que todos los requisitos funcionales estén cubiertos y correctamente rastreados en la matriz.
