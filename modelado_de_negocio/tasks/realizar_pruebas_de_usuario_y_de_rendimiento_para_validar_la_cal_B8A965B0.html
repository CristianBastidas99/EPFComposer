Task: Realizar pruebas de usuario y de rendimiento para validar la calidad del Escenario Vivido.
Realizar pruebas para asegurar la calidad, el rendimiento y la usabilidad del Escenario Vivido.
Purpose
Evaluar y validar la calidad del Escenario Vivido mediante pruebas de usuario y de rendimiento.
Relationships
RolesPrimary Performer:
  • Especialista en pruebas
Additional Performers:
InputsMandatory:
  • Escenario Vivido
    		Optional:
    • None
    Outputs
    • Escenario Vivido
      Main Description
      En esta tarea, se llevarán a cabo pruebas de usuario y de rendimiento para evaluar y validar la calidad del Escenario Vivido. Esto implica probar la funcionalidad, la usabilidad, la experiencia del usuario y el rendimiento del escenario simulado. Se evaluará la interacción, la respuesta del sistema, la carga de trabajo y otros aspectos relacionados con el rendimiento. Al realizar estas pruebas, se asegura que el Escenario Vivido cumpla con los estándares de calidad establecidos y brinde una experiencia satisfactoria y efectiva al usuario.
      Steps
      Revisar los requisitos del Escenario Vivido con el Analista de Negocio para entender los objetivos y funcionalidades esperadas.
      Crear un plan de pruebas que incluya escenarios de usuario y pruebas de rendimiento.
      Ejecutar pruebas manuales o automatizadas para validar la calidad del Escenario Vivido.
      Documentar cualquier problema o error encontrado durante las pruebas y reportarlos al Gerente de Proyecto y al Desarrollador FrontEnd.
      Trabajar con el Desarrollador FrontEnd para asegurar que los problemas encontrados sean resueltos adecuadamente y realizar pruebas de regresión para confirmar que los cambios no afecten a otras funcionalidades.
