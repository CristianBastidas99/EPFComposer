Task: Identificar las pruebas correspondientes a cada requisito
Asociar y especificar las pruebas necesarias para validar cada requisito funcional identificado.
Purpose
Determinar qué pruebas deben realizarse para verificar el cumplimiento de cada requisito funcional.
Relationships
RolesPrimary Performer:
  • Especialista en Pruebas
Additional Performers:
InputsMandatory:
  • Matriz de trazabilidad de requisitos
    		Optional:
    • None
    Outputs
    • Matriz de trazabilidad de requisitos
      Main Description
      En esta tarea, se identificarán y especificarán las pruebas correspondientes a cada requisito funcional del sistema. Se determinarán los tipos de pruebas adecuados (por ejemplo, pruebas de unidad, pruebas de integración, pruebas de aceptación) y se establecerán los criterios de prueba apropiados para cada requisito. Al identificar las pruebas correspondientes a cada requisito, se asegura una cobertura completa y adecuada de las funcionalidades del sistema.
      Steps
      Revisar la lista de requisitos funcionales del sistema.
      Identificar las pruebas que se deben realizar para verificar que cada requisito se cumpla adecuadamente.
      Registrar la relación entre cada requisito y las pruebas correspondientes en la matriz de trazabilidad.
      Asegurarse de que todas las pruebas necesarias estén identificadas y relacionadas con los requisitos correspondientes.
